Aluna — Containers

C4 Level 2 · deployable units and protocols

Data split

IndexedDB holds all health data (logs, symptoms, notes, settings). Neon holds identity (users/accounts), push subscription endpoints, and partner share snapshots only.

Auth model

Auth.js issues a JWT in an httpOnly Secure SameSite=Lax cookie. No OAuth tokens or session rows are persisted in Neon.

Push pipeline

Vercel Cron triggers /api/cron/reminders (batched, max ~200 users/run). The function reads subscriptions from Neon and calls the browser push service; the Serwist service worker shows the notification.

Hosting

App, API routes, and cron all deploy to Vercel. Database on Neon. Google OAuth client in GCP Console.
