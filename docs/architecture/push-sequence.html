Why Neon needs reminder data

IndexedDB is invisible to the server. To fire a push while the app is closed, the user must upload minimal schedule data (next period date, reminder preferences) when enabling push or saving settings.

Cron safety

Each cron invocation processes at most ~200 subscriptions — well within Vercel's function timeout. Pagination continues on the next hourly run if needed.

iOS caveat

Web push on iOS requires the PWA to be added to the Home Screen (iOS 16.4+). Document this in onboarding.
